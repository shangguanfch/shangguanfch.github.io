[Settings Sync](https://marketplace.visualstudio.com/items?itemName=Shan.code-settings-sync) is a good extension to synchronize settings, but when I use it (Shift+Alt+U), it shows "Sync: 未连接到网络或者连接GitHub失败。错误信息已打印到控制台", if you use windows, you should go to C:\Windows\System32\drivers\etc, and delete you own lines.
